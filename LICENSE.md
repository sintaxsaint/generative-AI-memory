## What Is Being Licensed

The intellectual property covered by this license includes the following original concepts as first documented by Elliot Readman in March 2026:

1. **The dual memory layer architecture** — the use of a lesser memory (staging layer) and core memory (weighted permanent knowledge base) working together in an AI system
2. **Evidence-based weight reinforcement** — storing concepts once and incrementing their weight upon repeated encounter rather than duplicating training data
3. **Evidence-based weight reduction** — reducing the weight of concepts not through passive time decay but through the accumulation of contradicting evidence
4. **Inference exclusively from core memory** — ensuring all model responses are generated only from reinforced, weighted knowledge rather than unverified new information
5. **Moral development through weight reinforcement** — building ethical understanding through repeated exposure to real-world evidence rather than hardcoded rules
6. **The lesser-to-core promotion mechanism** — the process by which new information is staged, compared, and integrated into core memory through reinforcement

These concepts collectively constitute the **Generative Memory architecture** and are the intellectual property of Elliot Readman regardless of the form in which they are implemented.
